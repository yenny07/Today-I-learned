## 10819 : 차이를 최대로
### 문제
https://www.acmicpc.net/problem/10819
### 코멘트


### 코드
```python
from itertools import permutations
import sys

N = int(sys.stdin.readline())
arr = list(map(int, sys.stdin.readline().split()))
answer = 0

for i in list(permutations(arr)):
    sum = 0
    for j in range(len(i) - 1):
        sum += abs(i[j] - i[j + 1])
    if sum > answer:
        answer = sum

print(answer)
```
