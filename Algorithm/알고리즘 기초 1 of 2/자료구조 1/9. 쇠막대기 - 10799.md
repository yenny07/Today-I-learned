### 코드
```python
from sys import stdin


bars = list(stdin.readline())
count = 0
stack = []

for i in range(len(bars)):
    if bars[i] == '(':
        stack.append('(')

    else:
        if bars[i-1] == '(':
            stack.pop()
            count += len(stack)
        else:
            stack.pop()
            count += 1

print(count)
```
