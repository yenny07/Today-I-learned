## 1463 : 1로 만들기
### 문제
https://www.acmicpc.net/problem/1463
### 코멘트
처음으로 만난 다이나믹 프로그래밍...!
메모이제이션이라는 용어를 오랜만에 들었다.
맞다 이렇게 하는 거였지... 살살 이해만 하고 내일 깊게 나만의 풀이과정을 적어봐야지.

### 코드
```python
from sys import stdin

N = int(stdin.readline())
d = [0] * (N+1)

for i in range(2, N+1):
    d[i] = d[i-1] + 1
    if i % 3 == 0:
        d[i] = min(d[i], d[i // 3] + 1)
    if i % 2 == 0:
        d[i] = min(d[i], d[i // 2] + 1)
print(d[N])
```
