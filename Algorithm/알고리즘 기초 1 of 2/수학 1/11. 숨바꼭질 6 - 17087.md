## 17087 : 숨바꼭질 6
### 문제
https://www.acmicpc.net/problem/17087
### 코멘트
### 코드
```python
from sys import stdin

def gcd(x, y):
    while y:
        x, y = y, x % y

    return x

N, S = map(int, stdin.readline().split())
A = list(map(int, stdin.readline().split()))

distances = list(set(abs(A[i] - S) for i in range(N)))
D = min(distances)

for i in range(len(distances)):
    D = gcd(distances[i], D)

print(D)
```
